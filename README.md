# connect_four/

The purpose of this project is to learn test driven development/

# Structure/
Board class:/
-has board size/
-initializes board/
-displays state/
-places pieces/
-checks win condition/

Player class:/
-has player names and piece color/
-makes moves/

Game class:/
-has current  player, game board, and game state/
-starts the game/
-switches players/
-handles player moves/
-checks for wind or draw/
-provides interface/
-handles user input/
-displays the board/
